    Karten

    Eine mit Karten gefüllte Rasteransicht im Inhaltsbereich wird mit einem <article class="grid"> Element eingeleitet. Jede Karte ist ein <section> Element innerhalb dieses Elternelements.

    Karten können einen Link enthalten. Die anklickbare Fläche des Links erstreckt sich über die gesamte Karte.

              <article class="grid">
            <section>
              <img src="https://picsum.photo/id/0/300/200" alt="">
              <div>
                <h2>Überschrift Karte 1</h2>
                <p>Beschreibung der Karte</p>
                <a href="#"></a>
              </div>
            </section>
          </article>

    Listen

    Ähnlich der Kartenansicht liefert die Listenansicht eine Übersicht gleichartiger Elemente. Jedes Element erstreckt sich über die gesamte Breite des Inhaltsbereichs und alle Elemente werden untereinander angezeigt.

    Elemente in der Listenansicht sind Tabellenzeilen (<tr>) in einer Tabelle (<table>). Der Haupttext kommt in die erste Spalte oder in die zweite Spalte, wenn zu dem Element ein Bild angezeigt werden soll. Das Bild kommt dann in die erste Spalte und die Überschrift des Haupttextes erstreckt sich dann über die ersten beiden Spalten.

    Alle weiteren Daten und Steuerelemente erscheinen ab der zweiten bzw. dritten Spalte

    Detailansicht

    Dies ist eine Blaupause für die Anzeige von detaillierten Daten eines einzelnen Elements.

    Die angezeigten Elemente werden zentriert im Inhaltsbereich ausgegeben und nehmen nicht die volle Breite des Inhaltsbereichs ein.

    Im einfachsten Fall werden die Daten als Description-List dargestellt. Dabei können Daten in Gruppen zusammengefasst werden, wenn die Gruppen jeweils eine eigene Description-List besitzen. Die Gruppen können eine Überschrift mit einem <h1> bis <h6> Element bekommen. Die Überschrift erfolgt aber in der Regel im Zentralbereich des Headers. 

            <dl>
          <dt>Name</dt>
          <dd>Paprika</dd>
          <dt>Anzahl</dt>
          <dd>8</dd>
          <dt>Änderung</dt>
          <dd><time>2025-02-01T17:06</time></dd>
        </dl>

    Änderung

    Trennlinien

    Nach einer Gruppe von Daten, die sich in der selben Description-List befinden, kann die folgende Zeilen mit einer Linie getrennt werden. Dazu müssen die Paare von dt und dd Elementen von einem div Element umgeben werden.

    Name
    Rote Paprika
    Allgemeiner Name
    Paprika
    Anzahl
    6

    Mehrere Definitionsterme für einen Wert

    Für einen Wert können in der Description-List mehrere Definitionsterme angegeben werden. Dabei wird der erste Term hervorgehoben und die nachfolgenden Terme unterhalb des ersten Terms angeordnet, während der darzustellende Wert als Description-Details Element vertikal zentriert auf der linken Seite erscheint.
